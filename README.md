    * 홍보 동영상(AutoPlay, 네비게이션 hover시 자동 멈춤)

    * 공지사항 롤링 구현
    * 사용자 입장 주 사용 메뉴 icon 배치

    * 슬라이드 Swiper plug-in 적용
